Сайт базы отдыха "FurTree" на основе фреймворка Django с адаптивным дизайном, использованием встроенной СУБД SQLite. Есть возможность просмотра сайта как локально, так и спомощью контейнера в Docker. Разработан функционал авторизации и регистрации на сайте, просмотр и фильтраця каталога домов, функция бронирования домов и услуг, просмотр и написание отзывов, разработана административная часть по шаблону Django.
HTML/CSS/JS + Django + Docker
